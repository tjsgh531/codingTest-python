1. 딕셔너리
    1-1 딕셔너리 메소드
    *딕셔너리리스트 : 특수 리스트라 생각하면 편함 일반 리스트로 바꿀려면 list()함수 사용해야함.
    딕셔너리.keys() : 키들은 모은 딕셔너리리스트를 반환
    딕셔너리.values() : 값들을 모은 딕셔너리리스트를 반환
    딕셔너리.items() : (키,값)형태의 튜플들을 모은 딕셔너리리스트 반환
    딕셔너리.clear() : 초기화
    딕셔너리.get(키) : 키와 매치되어있는 값을 반환
    키 in 딕셔너리 : 키값이 딕셔너리 안에 있으면 True 없으면 False 반환

    1-2 딕셔너리 특징
    Uniterable 객체 = 순서가 없음
    중복 허용 X

2. 다항연산자
    2-1 삼항연산자 
        : 실행 if 조건1 else 조건2
    2-2 한 줄에 사용하는 for
        : result = [ 실행문 for x in 리스트 ]
        : 리스트 내에서만 사용 가능!!!!!
         
        => 실행문을 실행시 x를 리스트에서 하나씩 꺼내면서 반복가능
        => 실행문이 삼항연산자(실행문 if 조건1 eles 조건2 )를 사용가능

        : 실행문 for x in 리스트 if 조건
        => 실행문을 for문으로 반복을 하는데 if 문으로 특정 x를 실행 못하게 막을 수 있음

        *따라서 "실행문 if 조건1 else 조건2 for x in 리스트 if 조건"이라는 혼종이 나올 수 있음
